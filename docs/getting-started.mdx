# Getting Started

This guide will help you quickly integrate Headless Select into your project. While the core library is framework-agnostic, we'll demonstrate a basic setup using each framework we support to illustrate the workflow.

## 1. Installation

First, install the wrapper for your chosen framework.

  <Tab active={true} label="React" icon={<SiReact />}>
    ```bash
    # Using npm
    npm install @verbpatch/react-select

    # Using yarn
    yarn add @verbpatch/react-select

    # Using pnpm
    pnpm add @verbpatch/react-select
    ```
